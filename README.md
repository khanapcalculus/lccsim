## Getting Started

This is a static website that works directly with GitHub Pages. Simply:

1. Clone the repository
2. Open `index.html` in a web browser
3. Navigate through the different math simulations

## GitHub Pages

The site is configured to work with GitHub Pages. To enable:

1. Go to your repository settings
2. Navigate to Pages
3. Select the `main` branch as the source
4. Your site will be available at `https://[username].github.io/lccsim/`
